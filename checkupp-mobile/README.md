# CheckUpp - Digital Health Passport

## Overview

**CheckUpp** is a digital health planner designed to empower users to manage their preventive healthcare proactively. It allows users to securely store health records, set reminders for health screenings, and organize essential health-related documentation. The app bridges the gap between clinician-controlled data and personal health records, giving users full control over their preventive health journey. With an adaptable design, **CheckUpp** can serve users across various health contexts and countries.

## Key Features

- **Prescription + Referral Wallet**: Securely store eScripts and referrals with a renewal reminder system.
- **Cancer Screening Reminders**: Set reminders for cervical, breast, bowel, prostate, lung, and skin cancer screenings.
- **Cardio-metabolic Screening Tracker**: Log and monitor blood pressure, weight, waist circumference, HbA1c, and cholesterol levels.
- **Pregnancy Check-Up Planner**: Manage checkups, blood tests, scans, and vaccinations during pregnancy.
- **Childhood Vaccination Reminders**: Follow vaccination schedules based on national guidelines.
- **Pre-Employment Check-Up Reminders**: Track necessary check-ups for various industries such as aviation, railway, and commercial driving.
- **Multi-Language Support**: Leverages Google Translate to deliver the app in the user's language of choice.

## Security and Privacy

- **Data Encryption**: All health data is encrypted and securely stored.
- **Two-Factor Authentication**: Enhanced security with optional two-factor authentication for account access.
- **Regular Backups**: Automatic data backups to ensure data integrity and accessibility.

## Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/colinmarklubembe/health-passport-mobile-app.git
   ```

2. Navigate to the project directory:

   ```sh
   cd health-passport-mobile-app
   ```

3. Install dependencies:

   ```sh
   npm install
   ```

4. Run the app:

   ```sh
   npx expo start -c
   ```

## Usage

1. Open the app on your mobile device.
2. Register or log in with your credentials.
3. Use the app to manage health records, set reminders, and access preventive health tools.
